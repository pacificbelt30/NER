# NER

## REQUIREMENTS
- python
- pipenv
- nltk dataset/model
- [みんなの自動翻訳＠TexTra®](https://mt-auto-minhon-mlt.ucri.jgn-x.jp/) or other translation services API account and key.

```
pipenv install
```

```python
python -m nltk.downloader popular
```
